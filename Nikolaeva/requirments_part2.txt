В программе используется подключение к БД Misrosoft SQL Server.
В переменную myServer записывается имя сервера, в переменную myDatabase имя базы данных, в которой заранее лежит собранный датасет.
В БД должно быть 2 таблицы: posts - содержит id постов, признаки из постов и классы, которым принадлежат эти посты для обучения моделей; 
test_posts - содержит id постов и классы для проверки работы обученной модели.

В переменной mySQLQuery хранится SQL запрос на извлечение данных из таблицы, имя которой указывается после SELECT * FROM.

В переменной access_token хранится токен для подключения к ВК через vk_api.

Все импортированные библиотеки должны быть установлены.
